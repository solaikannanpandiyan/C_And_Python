# Today problems: String Problems and Solutions -> Use Python

## 1. Check if a String is a Valid IP Address

**Description:** Verify if a given string follows the IPv4 format (x.x.x.x where 0 <= x <= 255).

**Example:**

```plaintext
Input: "192.168.1.1"
Output: True

Input: "256.100.1.1"
Output: False
```

## 2. Check if a String is a Valid Number

**Description:** Determine if a string represents a valid integer or decimal number.

**Example:**

```plaintext
Input: "123.45"
Output: True

Input: "12a.34"
Output: False
```

## 3. Compress a String

**Description:** Perform basic run-length encoding by replacing consecutive repeating characters with character count.

**Example:**

```plaintext
Input: "aaabbc"
Output: "a3b2c1"
```

## 4. Check if a String is a Rotation of Another

**Description:** Determine if one string is a rotation of another using concatenation.

**Example:**

```plaintext
Input: ("abcde", "deabc")
Output: True

Input: ("abc", "bca")
Output: True
```

## 5. Find All Substrings of a String

**Description:** Generate all possible substrings of a given string.

**Example:**

```plaintext
Input: "abc"
Output: ["a", "b", "c", "ab", "bc", "abc"]
```

## 6. Find the First Non-Repeating Character

**Description:** Identify the first character that appears only once in the string.

**Example:**

```plaintext
Input: "swiss"
Output: "w"
```

## 7. Sort the Words in a List in Lexicographical Order

**Description:** Sort a list of words in dictionary order.

**Example:**

```plaintext
Input: ["apple", "banana", "cherry"]
Output: ["apple", "banana", "cherry"]
```
